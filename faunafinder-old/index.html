What is fauna finder?

Fauna Finder is a mobile app developed by Penelope LTD, a group based in the University of York. The app is designed to help students and visitors identify and learn about the wildlife on campus. The app currently focuses on avian wildlife, with the intent to expand it to include other wildlife and animals in the future. The app acts as an identification tool to help students and visitors alike to find and learn more about the diverse campus wildlife. The app features a location screen that allows users to select from the available university campuses. Once a campus is selected, the app updates the list of animals accordingly. The app provides users with a long list of birds, their respective names, and corresponding illustrations. Each bird species comes with detailed information that includes its scientific name, physical description, interesting facts, and natural habitat.

Key features and benefits

Users can tap on a bird in the list to access its profile, which provides detailed information about the species. The illustrations in the main menu are designed to amplify the most pronounced features of the bird, aiding in its identification. The profile page includes several types of media, including photos, videos, and audio recordings of the bird in question. Additionally, users can learn about the bird's diet, location on campus as well as worldwide, and lots of other fun and interesting facts. The app's design is visually appealing and user-friendly, making it easy for anyone to navigate. The app is designed to be accessible to bird enthusiasts of all levels, from amateur birdwatchers to professional ornithologists. The app's comprehensive data on each bird species makes it a useful tool for research, education, and conservation efforts.

A look at the app

When users launch our app, they are presented with a user-friendly interface that allows them to easily navigate and explore information about birds in their selected campus location. The first step is for users to select the campus location they are interested in from the available options. This selection filters the bird data specific to the chosen campus, ensuring that users receive relevant and accurate information.


Once users have chosen their desired campus location, they are presented with a comprehensive list of birds that inhabit that particular area. The list provides an overview of the bird species, allowing users to browse through and find the birds they are interested in learning more about. Users can conveniently click on a specific bird from the list to access the individual bird's dedicated page.


On the bird's individual page, users are provided with detailed information about the selected bird species. The page is designed to offer an immersive experience, allowing users to explore different sections that provide valuable insights into the bird's characteristics, behavior, habitat, and more. Users can expand on each section to access in-depth information, enabling them to deepen their knowledge and understanding of the specific bird species they are interested in.


In addition to the bird-specific pages, our app includes an 'About the App' section. This section serves as a valuable resource for users to learn more about how to effectively utilize the app's features and functionality. It provides clear instructions, tips, and guidelines on navigating the app, accessing bird information, and maximizing their experience. Furthermore, the 'About the App' page also introduces users to the creators of the app, sharing insights into the team behind its development. Additionally, information regarding licensing and attributions is provided to ensure transparency and compliance with any necessary copyright or usage requirements.


By offering a user-friendly interface, easy campus selection, comprehensive bird lists, detailed individual bird pages, and informative sections, our app provides users with an enriching experience. Users can effortlessly explore and expand their knowledge about various bird species, their habitats, and the campus locations they inhabit, enhancing their appreciation and understanding of avian wildlife.

Our system

The system consists of an android user application, a desktop admin application and a database hosted on a remote server.


On the main user client, developed for Android, the user is presented with the home page containing a list of local avian wildlife species upon launching the application. From this page the user can change their location or view information about the application. Choosing a bird presents the user with a bird profile page containing the related information. Basing this application on android use allows the user to easily access the application while on the go.


A secondary guide admin client, developed for desktops, allows guides to create, read, update, and delete information from the database. Due to this client giving access to the application’s database, appropriate security measures need to be taken to ensure the validity of the user's intentions. This includes requiring an API key to access the client. Basing this application on desktop use allows for ease when editing information and adding content from file browsers.


Having an admin key on the admin client will give the required access for creating API keys and creating/updating locations.
